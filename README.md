# vim-profile

my vim profile for roaming.

Sets up [.vimrc](.vimrc) with perferred settings and installs the following:

plugin managers

- Vundle
- pathogen

plugins

- NERDtree
- ctrlp
- powerline
- numbers

## Installation

``` bash
curl https://raw.githubusercontent.com/jnyryan/vim-profile/master/install.sh | bash
```
